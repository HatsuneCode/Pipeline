## 全量下载当日文献/专利 ##

1.pubmed.py: 从 pubmed 上全量下载当日发表的文献，采用 Bio.Entrez API。

2.openAlex.py: 从 openAlex 上全量下载生物医药领域当日发表的文献，采用 pyalex API。

3.rxiv.py: 从 biorxiv/medrxiv 上全量下载当日发表的文献，采用 api.biorxiv。

4.lens.py: 从 Lens 上全量下载当日医学的专利，采用 api.lens。

5.serp.py: 从 SerpAPI 上全量下载医学生物学的谷歌 Patent 的专利，采用 SerpAPI。

sup1.filterByDeepseek.py: 通过 Deepseek 判断该文献是否在生物医药领域内。
